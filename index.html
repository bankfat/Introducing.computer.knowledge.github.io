Introducing Computer

เรียนรู้พื้นฐานเกี่ยวกับคอมพิวเตอร์และองค์ประกอบสำคัญ

คอมพิวเตอร์คืออะไร?

คอมพิวเตอร์ คือ อุปกรณ์ทางอิเล็กทรอนิกส์ (electronic device) ที่มนุษย์ใช้เป็นเครื่องมือช่วยในการจัดการข้อมูลที่อาจเป็น ทั้งตัวเลข ตัวอักษร หรือสัญลักษณ์ที่ใช้แทนความหมายในสิ่งต่าง ๆ โดยคุณสมบัติของคอมพิวเตอร์คือการที่สามารถกำหนดชุดคำสั่งล่วงหน้า หรือโปรแกรมได้

องค์ประกอบหลักของคอมพิวเตอร์

🖥️ Monitor (จอภาพ)

  • แสดงภาพ วิดีโอ เกม และทุกอย่างที่คอมทำงาน
  • เหมือน “หน้าต่าง” ของคอมพิวเตอร์

    • ⌨️ Keyboard (คีย์บอร์ด)

  • ใช้พิมพ์ข้อความ คำสั่ง เล่นเกม
  • มีปุ่มตัวอักษร ตัวเลข และปุ่มลัดต่างๆ

    • 🖱️ Mouse (เมาส์)

  • ใช้คลิก เลื่อน ลาก กดเลือกสิ่งต่างๆ บนจอ

    • 🔊 Speaker (ลำโพง)

  • ส่งเสียง เพลง เกม หนัง

    • 🖨️ Printer (เครื่องพิมพ์)

  • พิมพ์เอกสาร รูปภาพ ออกมาเป็นกระดาษ

    • 🔋 UPS (เครื่องสำรองไฟ)

  • สำรองไฟเวลไฟดับป้องกันคอมดับทันทีช่วยป้องกันไฟกระชาก

    • 🖥️ Case (เคส)

  • เป็น “กล่อง” ใส่อุปกรณ์ทั้งหมด
  • ช่วยระบายความร้อนและป้องกันอุปกรณ์เสียหาย

    • ⚡ Power Supply (พาวเวอร์ซัพพลาย)

  • จ่ายไฟให้ทุกชิ้นส่วนในคอม เปรียบเหมือน
    “หัวใจไฟฟ้า” ของเครื่อง
  • ถ้า PSU ไม่ดี: เครื่องดับ รีสตาร์ทเอง อุปกรณ์อาจเสียได้

    • 🧠 Mainboard / Motherboard (เมนบอร์ด)

  • เป็นแผงวงจรหลัก เชื่อมทุกชิ้นส่วนเข้าด้วยกัน
  • เช่น: CPU RAM การ์ดจอ SSD/HDD ทั้งหมดเสียบอยู่บนเมนบอร์ด

    • 🧠 CPU (ซีพียู)

  • สมองของคอม คำนวณและประมวลผลทุกอย่าง
    ยิ่งแรงเครื่องยิ่งเร็ว
  • เช่น: เปิดโปรแกรม เล่นเกม เขียนโค้ด คิดเลข

    • 🌪️ CPU FAN (พัดลมซีพียู)

  • ระบายความร้อน CPU ป้องกันเครื่องร้อนเกิน
  • ถ้าไม่มี: CPU ร้อนจัด เครื่องค้าง/ดับได้

    • 🟩 RAM (แรม)

  • เก็บข้อมูลชั่วคราวขณะใช้งาน “โต๊ะทำงาน” ของ CPU
  • RAM เยอะ: เปิดหลายโปรแกรมลื่นขึ้น เล่นเกมดีขึ้น ตัดต่อดีขึ้น

    • 💾 Harddisk / HDD

  • เก็บข้อมูลถาวร
  • เช่น: เกม รูป เพลง Windows
    ปัจจุบันนิยม SSD มากกว่าเพราะเร็วกว่า

    • 🎮 VGA Card / Graphics Card (การ์ดจอ)

  • ประมวลผลภาพ เกม วิดีโอ 3D ยิ่งแรง: FPS เกมยิ่งสูง
  • สำคัญมากสำหรับ: เล่นเกม ตัดต่อ ทำAI ทำกราฟิก
    ยิ่งแรง: FPS เกมยิ่งสูง

    • 🔈 Sound Card (การ์ดเสียง)

  • ประมวลผลเสียง
  • ปัจจุบัน: เมนบอร์ดส่วนใหญ่มีเสียงในตัวแล้ว

    • 💿 DVD ROM Drive

  • อ่านแผ่น CD/DVD สมัยนี้ใช้น้อยลง เพราะโหลดผ่านอินเทอร์เน็ตแทน

    • 📥 Reader All in One

  • อ่าน SD Card / เมมโมรี่การ์ด
  • ใช้กับ:กล้อง โทรศัพท์ โดรน

    • ที่มา: หลักการทำงานและวิธีเลือกใช้คอมพิวเตอร์

    ที่มา: หลักการทำงานของคอมพิวเตอร์

    ที่มา: อุปกรณ์คอมพิวเตอร์

    ประโยชน์ของคอมพิวเตอร์

    • ช่วยในการศึกษาและค้นคว้าข้อมูล
    • เพิ่มประสิทธิภาพในการทำงาน
    • ใช้ติดต่อสื่อสารผ่านอินเทอร์เน็ต
    • สร้างสื่อและความบันเทิง
    • รองรับการพัฒนาเทคโนโลยีและนวัตกรรม
